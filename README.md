# Мониторинг процесса `test`

Тестовое задание — bash-скрипт + systemd timer для мониторинга процесса `test`.

### Что реализовано
- Проверка каждую минуту через `systemd.timer`  
- Детекция рестартов процесса по смене PID  
- Запись рестартов в `/var/log/monitoring.log`  
- HTTPS-запрос на `https://test.com/monitoring/test/api` при живом процессе  
- Логирование недоступности мониторинг-сервера  
- Идемпотентность: при отсутствии процесса — ничего не делаем  
- Простая и надёжная установка через `install.sh`


Установка

- git clone https://github.com/Samurai-Champ100/test.git
- sudo ./install.sh
